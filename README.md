# fractol
A graphic program written in C during 42 School <br />
Linux version (tested on Ubuntu)
<br />
## Installation

```sh
$ git clone https://github.com/mnegura/fractol.git
$ cd fractol
$ make
$ ./fractol [fractol_name]
```
[fractol_names] are:
- **m || M** for Mandelbrot
- **j || J** for Julia
- **n || N** for Newton
- **t || T** for Titanic (a Mandelbrot alteration)
